# Gerenciamento de Tripulação (JavaScript)

Projeto do FreeCodeCamp focado no uso de loops para manipulação de dados em JavaScript.

## Funcionalidades
- Adição de membros sem IDs duplicados  
- Troca de posições na lista  
- Ordenação por prioridade (decrescente)  
- Filtro de astronautas aptos para EVA  
- Agrupamento em equipes menores  
- Exibição resumida da tripulação  

## Tecnologias
- JavaScript (ES6)

## Objetivo
Praticar lógica de programação utilizando loops e arrays.
